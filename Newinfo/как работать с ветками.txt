для того что бы создать новую ветку в репозитории 
git branch (name commit)

для того что бы переключится на новую ветку 
git checkout (названия ветки)

для того что бы связать после создания нашу ветку с удаленным репозиторием 
git push -u origin (названия ветки)

как нам связать изминения новой ветки с основной провести мердж
но для этого сначало необходимо вернутся на нашу основную ветку 
git checkout -
далее мы пишем git merge (названия ветки которую хотим замерджить)

так же что бы удалить ветку на локальной машине
git branch -d (name vetki)

удалить ветку с удаленного репозитория 
git push origin --deleted (name vetki)
